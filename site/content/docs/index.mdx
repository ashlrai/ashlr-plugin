ashlr is the open-source token ledger for AI coding tools. It replaces every Claude Code file read, grep, and edit with a lower-token alternative, tracks what it saves, and surfaces the ledger in a live status line.

**Mean savings: −79.5 % on files 2 KB and larger.** MIT-licensed. Zero telemetry. Works with Claude Code, Cursor, and Goose.

## Install in 30 seconds

```sh
claude mcp add ashlr -- npx -y ashlr-plugin
```

That is the entire install. See [Install](/docs/getting-started/install) for Cursor and Goose variants, plus how to verify the plugin is active.
