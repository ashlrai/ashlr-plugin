## Privacy commitments

- The plugin **never phones home** by default. Pro features only activate after you sign in.
- Cloud genome blobs are **end-to-end encrypted** with a per-team master key derived from X25519 keypairs the server can't decrypt.
- Telemetry is **opt-in** and never includes paths, source, or query content — see [docs/telemetry.md](https://github.com/ashlrai/ashlr-plugin/blob/main/docs/telemetry.md).
- **Zero training data** — your code is never used to train any model.
