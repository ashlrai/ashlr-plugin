# node-sdk reference subset

Subset of [github.com/vercel/ai](https://github.com/vercel/ai) at commit
`049801243cc791e0ba28910b8a638f0e05446bcd`, included for benchmark reproducibility.

32 TypeScript source files sampled from `packages/ai/src/` covering:
agent orchestration, embed, error types, generate-image, generate-object,
generate-speech, generate-text, telemetry, and utility modules.

No test files are included. Files ending in `.test.ts` or `.spec.ts` were excluded.
